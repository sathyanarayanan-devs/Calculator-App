## Simple Calculator

A **simple calculator** built with **HTML, CSS, and JavaScript**
It supports basic arithmetic operations and calculations, styled with clean, modern UI.

## Features

- Add, subtract, multiply, divide
- Clear button ('C') to reset input
- Responsive **grid layout** with visually distinct operator buttons
- Styled with hover and active button effects
- '=' evaluates the expression safely using JavaScript

## How to Use

1. Open index.html in your browser.
2. click the buttons to perform calculatons.

## Tech Stack

- HTML5 - Structure of the calculator
- CSS3 - Styling with grid layout and hover effects
- JavaScript (ES6) - Handles calculator logic and evaluation
